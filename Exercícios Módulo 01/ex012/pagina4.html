Para deixar a imagem transparente sem precisar mexer no editor de imagens.

O problema de se hospedar vídeos em seu
próprio computador (servidor local) é que
haverá um grande tráfego de dados, gerando 
um gasto muito grande, pois você irá 
prover os dados para as pessoas que estão 
usufruindo do seu site, ou seja, você fará
muito gasto com upload.

Para resolver este problema, podemos então usar links externos, assim não teremos gastos absurdos com dados de upload.
